# World-Text-Editor
As Funções:<br>
- Formatar em Italico; 
- Formatar em Negrito;
- Formatar em Underline;
- Aumentar a fonte;
- Diminuir a fonte; 
- Mudar cor do texto; 
- Mudar a cor do titulo; 
- Função para copiar o documento feito no editor(com titulo, e corpo do texto); 
- Alinhamento dos textos(á esquerda, centralizado, á direita, 
- Alterar a Fonte;
